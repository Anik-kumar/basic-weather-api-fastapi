This is simple weather status app.

In url add "/api/weather?city=XXXXXXX&state=YY&country=ZZ"

replace xxxxxx with city full name.

replace yy with state name. State name is 2 letter only. State name is optional.

replace zz with country name. Country name is 2 letter only. Default country is 'us'.
